# 介绍
<!-- {.md} -->

----
<!-- {.md} -->

`XU-UI` 是一款基于 `Vue.js 2.0` 的前端 UI 组件库，主要用于 PC 网站中后台产品的开发，目标定位是elementui等主流UI库的功能补充。

## 特性
<!-- {.md} -->

- 基于 `Vue` 的组件开发
- 背靠 npm 生态圈
- 项目发包基于 `travis` 集成，基于 `coveralls` 做测试覆盖率检测
